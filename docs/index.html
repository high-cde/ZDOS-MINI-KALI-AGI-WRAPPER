Installazione Rapida

1

Aggiorna Termux

pkg update && pkg upgrade -y
2

Installa le Dipendenze

pkg install -y nodejs python nmap curl wget openssl whois traceroute dnsutils tcpdump tshark bind-utils
3

Clona il Repository

git clone https://github.com/high-cde/ZDOS-MINI-KALI-AGI-WRAPPER.git
cd ZDOS-MINI-KALI-AGI-WRAPPER
4

Installa le Dipendenze Node.js

npm install
5

Avvia il Server API

node api/server.js

Comandi Principali

Scansione Base

zdos-mini scan basic example.com

Esegue una scansione Nmap di base con lookup WHOIS.

Recon Avanzato

zdos-mini scan advanced example.com

Scansione completa con banner grabbing e SSL audit.

Attack Surface Mapping

zdos-mini scan attack-surface example.com

Enumera sottodomini e scopre Cloud Bucket esposti.

OSINT Completo

zdos-mini osint full example.com

Raccoglie intelligence da multiple fonti pubbliche.

Audit Email

zdos-mini mail-audit email.txt

Analizza email per phishing e vulnerabilità.

Network Sniffing

zdos-mini sniffer capture wlan0 60

Cattura traffico di rete per 60 secondi.
